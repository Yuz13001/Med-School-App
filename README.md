在北美读医学院，做医生是很多学生的梦想。但是出于种种原因，这条路并不广为人知，不仅如此。中文网络上也流传着很多关于北美医学院的myth。借这个平台，写一篇文章，和大家聊一聊申请北美医学院的那些事儿，希望能抛砖引玉，让更多朋友了解这条路。
